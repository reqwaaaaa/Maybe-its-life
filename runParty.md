#### 出行前考试

  - 雷哥网刷题:https://gre.viplgw.cn/
  - ETS GRE官网刷题:https://www.ets.org/gre/test-takers/general-test/prepare.html
  - 2026年10月9日上午考第一场，26年底前四次考试内搞定：语文-150、数学-168、写作-4

GRE注意：
```
1.试题流程：AWA（30min） → Verbal 1 (12题/18min) → Quant 1 (12题/21min) → Verbal 2 (15题/23min) → Quant 2 (15题/26min)
2.Quantitative:内容相对简单（高中数学水平）z注意避免理解错误和计算失误。
3.Verbal: 每天背单词 + 做2-3组题（同步进行）。做题时总结经验：记录时间分配、易错题型、逻辑信号词等。单词量够大，Text Completion（填空）和Sentence Equivalence（句子等价）得分会很高。
  - 填空：靠词汇 + 上下文逻辑，注意微妙词义差异。
  - 阅读：重点把握作者意图和写作逻辑与结构
4.Analytical Writing:两篇作文，每篇30分钟，投入不必过多（每周3-4篇练习即可）
  - Issue Task（立论题）：针对一个观点发表自己的立场并论证。
  - Argument Task（驳论题）：分析给定论证的逻辑缺陷并指出不足。
```
- APS(26.12前搞定)：
流程：  
  1. 访问官网:https://www.aps.org.cn/zh/  
  2. 注册获取注册证明  
  3. 缴费并邮寄材料（手写签字）  
  4. 材料审核（约2周）  
  5. 面试通知 → 面试  
  6. 下载 DigZert 用于申请

APS注意：
```
1.其中的双语密封文件/公证件：内容上中英文对照:成绩单、毕业证/在读证明等文件必须由学校出具官方中英文版本(缺一不可);学校盖章:每页文件需加盖学校公章或教务处章(红色鲜章，复印章无效);密封形式上:将中英文文件放入学校官方信封，在信封封口处加盖学校公章(骑缝章)，确保未被私自拆开。
2.面谈结构分为笔试与面试两个环节。笔试环节上:题量1-5道，涵盖专业基础、核心课程及毕业论文相关内容;面试环节上:分为“基础问答”和“专业知识”考核，基础问答包括不限于“留学动机择校理由”等问题。
3.语言选择纯英,复习面要广而非深,重点关注一下英语专业名词和项目(基础科目和毕设)、知识点逻辑表达。
```

reference letter注意：
```
一、详细、与申请学科/项目一致贴合地介绍学生
二、letter编写大纲
1. Credentials of the author（推荐人资质）
   - 当前职位、研究方向、主要成就
   - 与云南大学/信息学院的关联
2. Connection with the candidate（关系建立）
   - 如何认识（课程、实验室、项目、学术指导等）
   - 具体合作时长和场景
3. Testimony（核心证据层 - 最重要，占40-50%篇幅）
   - 学术能力：GPA、核心课程表现、数学/算法基础
   - 研究创新：具体项目贡献
   - 工程能力：算法实现，系统落地
   - 品质：坚持、独立思考、文献阅读、团队/领导力
   - 挑战经历：实验失败、进度滞后时的应对
4. Small fault(s)（温和缺点 - 真实且转化为优势）
   - 示例：“有时在细节优化上投入过多时间，这也反映了他对工作质量的高标准。”
5. Prediction of a bright future（未来展望）
   - 明确学生适合攻读研究生（独立科研、发表潜力、跨领域应用）
   - 对申请项目（泛CS）的匹配度
6. 结尾
   - 强烈推荐 + 联系方式:
   Please feel free to contact me if further information is needed.
   Sincerely, Prof. X XX
   School of XXX Yunnan University
   Email: XXX@ynu.edu
三、俩/仨封信内容互补，避免重复; 每项优点必须有1个具体事例支撑; 550词左右（一页A4）; 使用样例中的正式信头（To Whom It May Concern / To the Admissions Committee），签名处留空给老师手签或电子签。
```
