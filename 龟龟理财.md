## 一、龟龟理财总纲

### 核心理念

`50w本金`龟龟流理财：

* 稳健资产守住底线
* 指数资产分享增长
* 股息资产建立现金流
* 时间完成复利

---

### 投资原则

1. 稳定优先：先保证生活安全，再追求收益
2. 长期主义：投资周期 ≥ 10年
3. 资产分层：不同资产承担不同功能

---

### 资产分层结构

#### 第一层：安全层（底线）

* 现金 / 货币基金 / 短债
* 作用：防风险 + 流动性

---

#### 第二层：股息层（核心）

* 高质量分红资产
* 稳定现金流来源

---

#### 第三层：增长层（进攻）

* 指数基金 / 优质成长股
* 获取长期收益

---

#### 第四层：机会层（增强）

* 烟蒂投资 / 事件驱动
* 提供超额收益

---

### 资金配置原则

* 10%：活钱
* 70%：股息资产（核心）
* 10%：烟蒂机会
* 10%：成长资产

---

## 二、商业模式筛选（“值得长期持有”的好公司）

1. 可理解性：能看懂行业 + 盈利逻辑收入来源清晰
2. 低资本开支：不需要持续大额投入维持 + 折旧/资本开支压力小
3. 现金收款模式：先收钱再服务 + 应收账款少
4. 垄断性 / 护城河：行业壁垒高 + 替代性低
5. 定价权：可以涨价而不失客户（抗通胀能力）
6. 弱周期性：不随经济大幅波动（如医疗）

* 量化代理变量：

*ROIC, 毛利率, 营收波动率, 资产周转率*

---

## 三、烟蒂投资（极端低估）

### 1. 资产垫（存量安全）

最优：*现金及等价物 - 总负债 > 市值*

次优：*现金及等价物 - 有息负债 > 市值*

普通：*流动资产 - 总负债 > 市值*

* 建模指标：

*M1 = (现金及等价物 - 总负债) / 市值*

*M2 = (现金及等价物 - 有息负债) / 市值*

*M3 = (流动资产 - 总负债) / 市值*

---

### 2. 正现金流（增量安全）

*自由现金流 = 经营现金流 - 资本开支 > 0*

* 建模指标：

*C1 = 自由现金流*

*C2 = 自由现金流 / 净利润*

*C3 = 现金流波动率（σ）*

---

## 四、价值兑现（价值闭环）

1. 分红（最优）：稳定派息 + 高股息率
2. 回购 + 注销：提升每股价值
3. 清算 / 并购：资产变现

* 建模指标：

*D1 = 股息率*

*D2 = 分红率*

*D3 = 分红连续年数*

*D4 = 回购强度（回购金额 / 市值）*

---

## 五、需注意（策略约束）

1. 烟蒂投资 ≠ 长期持有，本质是套利
2. 卖出：价格回归价值 / 安全边际消失
3. 适合市场：港股、美股小盘
4. 组合原则：

* 大部分资金 → 可持续股息
* 小部分资金 → 烟蒂机会

---
## 六、初步建模

### 1. 总体模型（多因子框架）

*TotalScore = w1·Dividend + w2·Margin + w3·Cashflow + w4·Business + w5·Event*

其中：

*w1=0.30, w2=0.25, w3=0.20, w4=0.15, w5=0.10*

---

### 2. 各因子建模（优化版）

#### （1）股息因子（Dividend）

*Dividend = 0.25·norm(D1) + 0.2·norm(D2区间) + 0.2·norm(D3) + 0.2·norm(增长率) + 0.15·norm(FCF覆盖率)*

逻辑：
* 生命周期理论 + 可持续分红

---

#### （2）安全边际因子（Margin）

*Margin = σ(α1·M1 + α2·M2 + α3·M3)*

其中 σ 为 sigmoid：

*σ(x) = 1 / (1 + e^{-x})*

逻辑：
* 避免极端值影响

---

#### （3）现金流因子（Cashflow）

*Cashflow = 0.3·sign(C1) + 0.25·norm(C2) + 0.25·(1/σ(C3)) + 0.2·norm(连续性)*

逻辑：
* 现金流折现模型
* 盈余质量理论

---

#### （4）商业模式因子（Business）

*Business = 0.3·norm(ROIC) + 0.25·norm(毛利率) + 0.2·(1/波动率) + 0.15·norm(周转率) + 0.1·行业因子*

逻辑：
* 竞争优势理论（护城河）
* 产业组织理论

---

#### （5）事件驱动因子（Event）

*Event = Σ(事件权重 × 发生概率)*

逻辑：
*P(event) = Logistic(Xβ)*
或使用：
* Random Forest
* XGBoost

数据来源：公告、新闻（NLP）

---

### 3. 数据挖掘建模流程

#### Step 1：数据构建

*X = [Dividend, Margin, Cashflow, Business, Event]*

*y = 未来收益（1年 / 3年）*

---

#### Step 2：特征处理

* 标准化：*z = (x - μ) / σ*
* 去极值（Winsorize）
* 缺失值填补

---

#### Step 3：模型训练

基础模型：

*ŷ = f(X)*

算法：
* 线性模型（解释性）
* 随机森林（非线性）
* XGBoost（最优）

---

#### Step 4：因子回测（核心）

*IC = corr(因子, 收益)*

*IR = IC / std(IC)*

---

#### Step 5：组合构建

按 TotalScore 排序：

* Top 10% → 买入
* Top 20% → 候选

---

#### Step 6：动态优化

根据市场状态调整权重：

* 熊：提高 Margin 权重
* 牛：提高 Business 权重

---

### 4. 风险控制模块

#### （1）价值陷阱过滤

*Decline = 营收趋势斜率*

若 < 0 → 降权

---

#### （2）财务风险

* 资产负债率
* 利息覆盖倍数

---

#### （3）行业过滤

剔除：强周期 + 高杠杆行业

---

## 七、总结

> 本体系 = *股息*（现金流）+ *烟蒂*（安全垫）+ *事件*（催化）

### 脚本

功能：
1. 导入股票财务数据CSV
2. 计算多因子评分（股息、安全边际、现金流、商业模式、事件驱动）
3. 输出总评分和排名
4. 梯度提升预测未来收益

使用：
1. 准备CSV文件：stocks.csv，列包括：

``` python
import pandas as pd
import numpy as np

# 工具函数

def sigmoid(x):
    """Sigmoid函数，用于安全边际极值压缩"""
    return 1 / (1 + np.exp(-x))

def normalize(series):
    """z-score标准化"""
    return (series - series.mean()) / (series.std() + 1e-6)

def inverse_vol(x):
    """波动率倒数"""
    return 1 / (x + 1e-6)

# 读取数据
df = pd.read_csv("stocks.csv")

# 基础计算
df["FCF"] = df["operating_cashflow"] - df["capex"]

# 因子1：股息
df["D1"] = df["dividend"] / df["market_cap"]  # 股息率
df["D2"] = df["payout_ratio"]
df["D3"] = df["dividend_years"]
df["D4"] = df["dividend_growth"]
df["D5"] = df["FCF"] / (df["dividend"] + 1e-6)

df["DividendScore"] = (
    0.25 * normalize(df["D1"]) +
    0.2 * normalize(df["D2"]) +
    0.2 * normalize(df["D3"]) +
    0.2 * normalize(df["D4"]) +
    0.15 * normalize(df["D5"])
)

# 因子2：安全边际
df["M1"] = (df["cash"] - df["total_liabilities"]) / df["market_cap"]
df["M2"] = (df["cash"] - df["interest_debt"]) / df["market_cap"]
df["M3"] = (df["current_assets"] - df["total_liabilities"]) / df["market_cap"]

df["MarginRaw"] = 0.5 * df["M1"] + 0.3 * df["M2"] + 0.2 * df["M3"]
df["MarginScore"] = sigmoid(df["MarginRaw"])

# 因子3：现金流
df["C1"] = np.sign(df["FCF"])
df["C2"] = df["FCF"] / (df["net_income"] + 1e-6)
df["C3"] = df["fcf_std"]

df["CashflowScore"] = (
    0.3 * df["C1"] +
    0.25 * normalize(df["C2"]) +
    0.25 * normalize(inverse_vol(df["C3"])) +
    0.2 * normalize(df["FCF"])
)

# 因子4：商业模式
df["BusinessScore"] = (
    0.3 * normalize(df["roic"]) +
    0.25 * normalize(df["gross_margin"]) +
    0.2 * normalize(inverse_vol(df["revenue_std"])) +
    0.15 * normalize(df["asset_turnover"]) +
    0.1 * normalize(df["revenue"])
)

# 因子5：事件驱动
df["E1"] = df["buyback"] / df["market_cap"]
df["EventScore"] = normalize(df["E1"])

# 总评分
df["TotalScore"] = (
    0.30 * df["DividendScore"] +
    0.25 * df["MarginScore"] +
    0.20 * df["CashflowScore"] +
    0.15 * df["BusinessScore"] +
    0.10 * df["EventScore"]
)

df = df.sort_values(by="TotalScore", ascending=False)

# 输出前20个股票评分
print(df[["ticker","TotalScore","DividendScore","MarginScore","CashflowScore","BusinessScore","EventScore"]].head(20))

# 保存
df.to_csv("scored_stocks.csv", index=False)

try:
    from xgboost import XGBRegressor
    # 假设CSV里有列future_return
    if 'future_return' in df.columns:
        X = df[["DividendScore","MarginScore","CashflowScore","BusinessScore","EventScore"]]
        y = df["future_return"]
        model = XGBRegressor(objective='reg:squarederror')
        model.fit(X, y)
        df["PredictedReturn"] = model.predict(X)
        df = df.sort_values(by="PredictedReturn", ascending=False)
        print("\n机器学习预测排序Top20:")
        print(df[["ticker","PredictedReturn"]].head(20))
except ImportError:
    print("未安装xgboost，跳过机器学习预测")
```
